# wezzard-skills

A Claude Code skill marketplace by WeZZard.

## Plugins

### amplify

WeZZard's Claude Code skills: brainstorming, planning, execution, error recovery.

### skill-kit

Tools for auditing and improving Claude Code skill definitions.
